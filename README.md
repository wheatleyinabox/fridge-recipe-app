# fridge-recipe-app

## Enviroment Setup
2. Download Android Studio
3. Clone repo into VSCode
5. Create an Virtual Device within Android Studio's Virtual Device Manager (recommended: Android 15.0 VanillaIceCream)
### For Frontend...
```
cd fridge-app
npm install
npm run android
```
### For Backend...
```
cd recipes
npm run start
```
